How to Download Python ?

ขั้นตอนกาติดตั้ง Python

1.กดที่รูปภาพด้านบนเพื่อไปหน้า Download

2.กด Download Python

3.รอ Download ให้เสร็จ

4.กดติดตั้งโปรแกรม

5.กดคลิกที่ Add Python to PATH แล้วกด Install Now

6.รอสักครู่ โปรแกรมกำลังติดตั้ง

7.หลังจากที่การติงตั้งเสร็จสิ้นแล้ว คลิก “Close” เพื่อเสร็จสิ้นการติดตั้งภาษา Python

8.เราได้ติดตั้งภาษา Python เรียบร้อยแล้วและคุณพร้อมที่จะเขียนโปรแกรมในภาษา Python

9.ต่อไปเราจะทำการตรวจสอบการติดตั้ง โดยให้คุณเปิด Command line ขึ้นมา ใช้คำสั่ง python และตามด้วยที่อยู่ของไฟล์ดังตัวอย่างข้างล่างนี้ c:\path\file_name.py

10.คลิกเปิดโปรแกรม Python IDLE

11.เปิดโปรแกรมแล้ว ให้พิมพ์คำสั่ง print("HELLO WORLD") กดปุ่ม Enter

12.เมื่อได้ผลลัพธ์ดังนี้แล้วถือว่าการติดตั้งเสร็จสมบูรณ์ คุณพร้อมที่จะเขียนโปรแกรมแล้ว
